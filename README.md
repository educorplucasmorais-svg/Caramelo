# Caramelo
Projeto de Proteção Animal e soluções ESG

## Funcionalidades

### Iniciar Sessão (Login)

O sistema inclui uma página de login para autenticação de usuários.

**Arquivo:** `src/login.html`

**Características:**
- Interface responsiva e amigável
- Validação de formulário do lado do cliente
- Validação de formato de e-mail
- Gerenciamento de sessão usando sessionStorage
- Mensagens de erro e sucesso em português

## Como Usar

1. Abra o arquivo `src/login.html` em um navegador web
2. Insira seu e-mail e senha
3. Clique em "Iniciar Sessão"
